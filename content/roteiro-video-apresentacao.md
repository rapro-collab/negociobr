## 🎬 Opções de Produção

### Opção A: 100% Capturas de Tela (Mais Rápido)
**Ferramentas**: OBS Studio (gravação) + CapCut (edição)

**Passo a passo**:
1. Gravar tela navegando pelo site (homepage, planos, perfil de empresa)
2. Capturar prints das artes HTML (abrir no browser e screenshot)
3. Editar no CapCut: adicionar transições, texto, música
4. Adicionar narração (voz humana ou IA)
5. Exportar em 9:16 (1080x1920)

**Tempo estimado**: 2-3 horas

---

### Opção B: Mix de Capturas + Animações (Mais Profissional)
**Ferramentas**: OBS Studio + Canva (animações) + CapCut

**Passo a passo**:
1. Gravar capturas de tela do site
2. Criar animações no Canva (usando templates de Reels)
3. Usar as artes HTML como base para animações
4. Editar no CapCut: sincronizar música, adicionar legendas
5. Adicionar narração profissional (voz humana)
6. Exportar em 9:16 (1080x1920)

**Tempo estimado**: 4-5 horas

---

### Opção C: Vídeo com Rosto (Mais Pessoal)
**Ferramentas**: Smartphone + Tripé + OBS (gravação de tela)

**Passo a passo**:
1. Gravar rosto falando (formato vertical)
2. Gravar capturas de tela do site
3. Editar: intercalar rosto + capturas
4. Adicionar legendas e texto
5. Adicionar música de fundo
6. Exportar em 9:16 (1080x1920)

**Tempo estimado**: 5-6 horas
